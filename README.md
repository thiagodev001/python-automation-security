# Automação de Segurança e Recuperação de Dados com Python

Este projeto foi desenvolvido como parte da simulação de carreira em Cibersegurança da **AIG (via Forage)**. O objetivo principal é demonstrar a aplicação da lógica de programação em Python para a resolução de problemas críticos de segurança e automação de processos.

## 目的 (Objetivo)
Desenvolver um script capaz de realizar a recuperação de arquivos criptografados por um simulado de Ransomware, utilizando técnicas de força bruta ética (bruteforce) através do cruzamento de chaves em dicionários de dados.

## 🛠 Tecnologias e Conceitos Utilizados
- **Linguagem:** Python 3.x
- **Lógica de Programação:** Estruturas de repetição (`for`, `while`) e condicionais (`if/else`).
- **Manipulação de Arquivos:** Leitura e processamento de listas de strings (wordlists).
- **Tratamento de Exceções:** Garantia de execução contínua do script durante a busca.

## 🚀 Como Funciona
O script recebe um arquivo criptografado e uma lista de chaves potenciais (ex: `rockyou.txt`). Ele automatiza a tentativa de descriptografia, validando cada chave até encontrar a correta, eliminando a necessidade de intervenção manual e aumentando a eficiência da resposta ao incidente.

## 📈 Resultados obtidos
- Redução do tempo de recuperação de ativos críticos.
- Demonstração de viabilidade técnica para automação de tarefas repetitivas em operações de TI.
- Aplicação de boas práticas de escrita de código (PEP 8).
